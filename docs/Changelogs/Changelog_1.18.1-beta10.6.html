Changelog for Chromamancy 1.18.1-beta10.6

Made several changes to summoned mobs, added the new Iridescent exclusive spell, Enthrall, and made more quality-of-life fixes!

Added

Added: Enthrall - New Iridescent exclusive spell: turns an existing mob into a summoned one with infinite duration.
Added: All summoned and enthralled mobs will now follow the caster, and will no longer be hostile to random players/entities.

Changed

Changed: Lock On - No longer a single level spell, and now has same and opposite class effects
Changed: Teleport - Changed the texture of the spell to better show the color at a quick glance
Changed: Double Summon - No longer exclusive to Iridescent Wizards
Changed: Capture Mob - Can no longer be used to capture summoned mobs, but will work with Enthralled ones.
Changed: Summon Phantom - Duration doubled to account for very slow attack cycles

Other

Fixed: Spells can now be cast when stuck in a ceiling or otherwise small area that forces a crouch
Fixed: Magical wolves now no longer get stuck when riding on the caster's head when the caster enters water.
Fixed: Reaching level 100 in a class should now properly display the name of the achiever in the chat
Fixed: Fixed missing tooltips on the capes
Fixed: Capture Mob - Should now correctly display the name of the mob contained within
