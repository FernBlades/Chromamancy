Changelog for Chromamancy 1.18.1-beta10.9

Added an Elytra robe enchantment, tweaked Health Boost and Regeneration, and improved the Flight spell with a bug fix and enchantment support!

Added

Added: Elytra Robe Armor Enchantment - Allows Elytra flight with the Wizard robes, higher levels reduce mana cost

Changed

Changed: Health Boost - Removed cooldown and increased duration, since recasting it will remove any hearts gained from the spell
Changed: Regeneration - Reduced the maximum level of Regeneration at higher levels, and changed some class bonuses.
        Previously, at level 5 this spell would heal so fast there was little need to create Heal. This change is aimed at giving both spells a use case
Changed: Flight - Now supports the Cost Reduction spell enchantment
Changed: Health Boost Ally - Same changes as Health Boost
Changed: Regeneration Ally - Same changes as Regeneration

Other

Fixed: Upgrades on the Flight spell now correctly reduce mana cost
Fixed: Chromatic Solution recipe
Fixed: Mana Master, Health Master, and Damage Master advancements should no longer trigger early
