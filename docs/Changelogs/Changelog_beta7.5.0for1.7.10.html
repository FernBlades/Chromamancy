Changelog for Spellcraft beta7.5.0for1.7.10

Added much simpler and appropriate recipes for the magical dusts, significantly revamped the berry crops, and added AgriCraft support.

Added

Dust Primer
        - Used as a base in new dust recipes.
        - Created by smelting a Mundane Potion

Agricraft Addon
        - Allows Spellcraft berries to be grown using Agricraft's crop sticks.

Magical Farmland
        - Created by using any dust on normal farmland
        - Used as a base for growing berries
        - Visually shows the mulches used on it

Infused Saddle
        - Can be used on any pet entity to let them ride on your head and dismount.

Berry Seeds
        - Now registered as seeds in the OreDictionary, so they should work in recipes that use any seeds.

Changed

Biome Conversion
        Fixed: No longer causes internal server error when cast.

Dust Recipes
        Changed: Have been completed redone, much simpler now.

Berry Crops
        Changed: No longer grow on a fixed time, they now grow based on random ticks.
        Changed: Now require Magical Farmland as a base

Tier 1 Berry Seeds
        Removed: No longer drop from tall grass
        Added: Now have crafting recipes

Removed

Nature's Blessing
        Temporarily removed until its new function can be implemented.
