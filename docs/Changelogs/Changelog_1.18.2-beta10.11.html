Changelog for Chromamancy 1.18.2-beta10.11

Infinite mana calls you, if you dare delve that far into Chromatic magic! This update introduces the Overflowing Chalice, which grants infinite mana when worn! Additionally, this update adds cross-mod support with several mods to allow for more automation.

Added

Added: Overflowing Chalice - Grants infinite mana when worn, but expensive to craft!
Added: Greater Mana Potions - Restore 50% of your max mana instead of the usual 20%
Added: Singularities - Crafting ingredients for Overflowing Chalice, see below for details
Added: Chromatic Fish - You can now make magical cats! Magical Cats are unkillable, and will ride on your head if you fly away (while they are not sitting)

Added: Mod support for Industrial Foregoing:
        - Berries are now compatible with the Plant Gatherer

Added: Mod support for Thermal Series:
        - Added recipes for pulverizing berries to tiny dust
        - Added recipes for pressing dust blocks into gems
        - Added recipes for the Multiservo Press to convert between dust blocks, dust, and tiny dust.

Added: Mod support for Avaritia:
        - Added Singularities for each dust block (requires 1000 blocks)
        - Added recipe for the Overflowing Chalice

Changed

Flight
        Fixed: Mana is no longer erroneously regenerated when the spell is disabled
        Fixed: No longer costs mana when cast while the Amarultimate berry effect is active

Changed: Touch of Life - will now also attempt to bonemeal the block below the target, if the target is invalid (Useful for growing fruits on trees)

Fixed: Capture Mob - Should now work on mobs riding other mobs or sitting down

Fixed: Attempted to fix the "failed spell" sound so it will play at the caster instead of directly on any nearby players
Changed: Spells with only a single level will now be cast when holding sneak/shift
