Changelog for Spellcraft beta6.12.0for1.6.4

This update takes another stab at balancing mana regen by hitting the Chalice of Harmony and the tools, which should now be usable again.

Added

Mana Regen potion effect:
        Not yet implemented, will be used in the future.

Changed

Locate Ores:
        Removed: Class Bonus Mana Cost: -10
        Added: Class Bonus Effect Level: +1
        Changed: Opposite Class Mana Cost: +15 to +30

Flight:
        Fixed: Turbo mode is working now.

All gem staves:
        Fixed: Updated damage tooltip for clarity.

Gem Tools:
        Changed: Mana Cost: No longer a percentage. Now cost is max mana / 1000, with a minimum 1 mana cost. Still increases with harvest level of block.
        Changed: Harvest Level: Raised to 5 from 3. Should now be able to harvest cobalt and ardite.
        Changed: Durability: Maximum durability raised to 1562 from 256.

Upgrades:
        Mana Boost:
                Changed: Each upgrade gives +100 maximum mana, up from +10.
                Changed: Maximum upgrades: Lowered to 19 from 190.

        Damage Boost:
                Changed: Maximum upgrades: Lowered to 4 from 9.

Berry Crops:
        Changed: Higher tier berry crops will now drop more berries than lower tiered ones.

Chalice of Harmony:
        Removed: No longer provides a passive -5s to mana regen time.
        Added: Uses stored mana potion to provide -5s to mana regen time.
        Craft with mana potions to add to the stored amount, can hold 8 potions worth.
