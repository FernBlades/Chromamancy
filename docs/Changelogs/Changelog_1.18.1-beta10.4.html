Changelog for Chromamancy 1.18.1-beta10.4

Added several new spells to help balance out the number of spells in each color!

Added

Added: Tame - Has a percent chance to tame the targeted creature, if it can be tamed
Added: Slow Falling - Fall slower and prevents fall damage while active
Added: Ironskin - Increases armor while active
Added: Slow Falling Ally - Rosacian version of Slow Falling
Added: Ironskin Ally - Rosacian version of Ironskin
Added: You can now compress dust blocks into gems using the Immersive Engineering Metal press, and a Chromatic Gem Mold

Other

Fixed: Restored the ability to disenchant robes using blank armor tomes
